# What is EquiLens

EquiLens is a predictive reliability monitoring system for deployed machine learning models.

Most monitoring tools tell you when something has already gone wrong.
EquiLens forecasts when something is about to go wrong — before the alert fires,
before the user is affected, before the damage accumulates.

It works entirely from prediction outputs. No model weights. No training data.
No access to model internals. Just the confidence scores and probability distributions
that any classifier produces as part of normal operation.

This makes it applicable to any deployed model — including models you didn't build,
don't own, and cannot inspect.
